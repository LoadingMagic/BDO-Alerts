## Features

**Timers & Alerts**
- World Boss timers with optional text-to-speech reminders.
- Daily and weekly reset timers, plus maintenance schedules for all regions.
- Golden Pig Cave spawn tracking with real-time status updates.
- Active coupon codes with one-click copy functionality.

**Market & Gameplay**
- Central Market data including prices, stock levels, and price history.
- Comprehensive grind spot database with silver-per-hour estimates.
- Built-in grind timer with customizable presets and session tracking.
- Cron Stone cost calculator for enhancement planning.
- Lightstone set builder with effect combinations and recommendations.

**Community**
- Player and guild search with detailed profiles and statistics.
- Leaderboards for tracking top players and guilds.
- Guild recruitment board for posting and browsing advertisements.
- Aggregated news feed from official Pearl Abyss announcements.
- Community wiki with guides, tips, and strategies.
- Class tier lists with PvE and PvP rankings.

**For Developers**
- Free RESTful API with JSON endpoints for all game data.
- No authentication required for public endpoints.
- Fast response times with high availability and continuous updates.
